# Modulo — Excel per Data Analysis

## Panoramica del Modulo

Questo modulo ha coperto Excel come strumento fondamentale per l'analisi dati, partendo dalle funzioni base fino ad arrivare a tecniche avanzate di trasformazione e modellazione dati. Nel corso di quattro settimane intensive più una Build Week di gruppo, ho costruito competenze complete nell'uso di Excel moderno: dalle formule logiche alle tabelle Pivot, da Power Query per la pulizia dati a Power Pivot per la creazione di modelli relazionali con DAX.

## Obiettivi del Modulo

- Padroneggiare le funzioni Excel fondamentali: logiche, di ricerca, conteggio e somma condizionale
- Creare report professionali con tabelle Pivot e grafici
- Utilizzare Power Query per importare, pulire e trasformare dati da fonti multiple
- Costruire modelli dati relazionali con Power Pivot e misure DAX
- Applicare tutte le competenze in un progetto reale di analisi COVID-19

## Settimane del Modulo

### Week 1 — Excel Fundamentals & First Git Steps

**Argomenti trattati:**
- Gestione dati: testo in colonne, rimozione duplicati
- Funzioni logiche: `SE`, `E`, `O`, `XOR`, `NON`
- Funzioni di conteggio e somma: `CONTA.SE`, `SOMMA.SE`, `SOMMA.PIÙ.SE`
- Ricerca e indicizzazione: `CERCA.VERT`, `CERCA.X`, `INDICE`, `CONFRONTA`
- Creazione e personalizzazione grafici
- Introduzione a Git: `init`, `add`, `commit`, `push`, `branch`, `merge`

**Competenze sviluppate:**
- Automatizzare calcoli ripetitivi con formule
- Creare mapping e lookup tra tabelle
- Visualizzare dati con grafici appropriati
- Tracciare il lavoro con Git e VSCode

**Sfide superate:**
- Calcoli IVA e concatenazione colonne su dataset fatture
- Mapping punteggi → giudizi con `CERCA.VERT`
- Logiche condizionali annidate per calcoli complessi

### Week 2 — Layout di Stampa & Tabelle Pivot

**Argomenti trattati:**
- Layout di stampa: margini, orientamento, area di stampa, interruzioni pagina
- Intestazioni e piè di pagina dinamici
- Creazione e gestione tabelle Pivot
- Filtri, righe, colonne e valori nelle Pivot
- Campi calcolati e formattazione condizionale
- Grafici da pivot e scatterplot con trendline

**Competenze sviluppate:**
- Rendere fogli Excel presentabili e stampabili
- Sintetizzare dataset complessi in pochi clic
- Creare report multi-foglio con layout ordinato
- Analizzare performance con filtri dinamici

**Sfide superate:**
- Analisi vendite per categoria e punto vendita
- Integrazione dati social media e calcolo engagement
- Report completi con insight divisi per pagina

### Week 3 — Power Query & Data Transformation

**Argomenti trattati:**
- Importazione dati da CSV e tabelle esterne
- Pulizia: rinomina colonne, rimozione righe, formati corretti
- Colonne calcolate: ricavi, prezzi scontati, quarter, year-month
- Merge e append tra tabelle con chiavi comuni
- Raggruppamenti per totali e medie
- UnPivot e ristrutturazione dataset

**Competenze sviluppate:**
- Costruire pipeline di trasformazione riutilizzabili
- Gestire dataset più grandi dei limiti Excel tradizionale
- Integrare multiple fonti in un dataset unico
- Controllare qualità e distribuzione dei dati

**Sfide superate:**
- Preparazione dataset AdventureWorks puliti e coerenti
- Gestione colonne con spazi, formati sbagliati, duplicati
- Integrazione tabelle product, sales, region, reseller, targets

### Week 4 — Power Pivot & Advanced Excel

**Argomenti trattati:**
- Attivazione e utilizzo Power Pivot
- Modelli dati con relazioni uno-a-molti
- Gerarchie e slicer per navigazione dati
- Misure DAX: classifiche dinamiche, top N, confronti con medie
- Funzioni Excel 365: `CERCA.X`, `FILTRO`, `DIVIDI.TESTO`, `SEQUENZA`
- Tabelle pivot avanzate su fonti integrate

**Competenze sviluppate:**
- Trasformare Excel in uno strumento BI completo
- Collegare tabelle in modelli relazionali
- Creare dashboard interattive con slicer e parametri
- Combinare Power Query + Power Pivot per analisi end-to-end

**Sfide superate:**
- Analisi dataset mondiali con confronti contro medie globali
- Classifiche dinamiche per indicatori selezionati
- Progetto finale: strutture ricettive e prezzi medi

### Build Week — Analisi COVID-19 Italia 2020–2025

**Progetto di gruppo (Team F):**
- Integrazione dataset COVID (Protezione Civile) e vaccinazioni (Open Data)
- Pulizia e normalizzazione entità: date, regioni, campi
- Costruzione KPI: contagiati/guariti, contagiati/deceduti, contagiati/vaccinati
- Dashboard stampabili A4 con layout professionale

**Deliverable:**
- Modello Power Pivot con relazioni tra tabelle dimensione e fatto
- Classifiche regionali 2020–2025 per contagiati, guariti, deceduti
- Rapporti percentuali per analisi severità/efficacia
- 6 fogli output: Contagiati, Guariti, Deceduti, Maschera, Rapporti, Efficacia Vaccini

# Module — Excel for Data Analysis

## Module Overview

This module covered Excel as a fundamental tool for data analysis, starting from basic functions up to advanced data transformation and modeling techniques. Over four intensive weeks plus a group Build Week, I built comprehensive skills in modern Excel: from logical formulas to Pivot Tables, from Power Query for data cleaning to Power Pivot for building relational models with DAX.

## Module Objectives

- Master fundamental Excel functions: logical, lookup, conditional counting and summing
- Create professional reports with Pivot Tables and charts
- Use Power Query to import, clean, and transform data from multiple sources
- Build relational data models with Power Pivot and DAX measures
- Apply all skills in a real COVID-19 analysis project

## Module Weeks

### Week 1 — Excel Fundamentals & First Git Steps

**Topics covered:**
- Data management: text-to-columns, remove duplicates
- Logical functions: `IF`, `AND`, `OR`, `XOR`, `NOT`
- Counting and summing: `COUNTIF`, `SUMIF`, `SUMIFS`
- Lookup and indexing: `VLOOKUP`, `XLOOKUP`, `INDEX`, `MATCH`
- Creating and customizing charts
- Introduction to Git: `init`, `add`, `commit`, `push`, `branch`, `merge`

**Skills developed:**
- Automating repetitive calculations with formulas
- Creating mappings and lookups between tables
- Visualizing data with appropriate charts
- Tracking work with Git and VSCode

**Challenges overcome:**
- VAT calculations and column concatenation on invoice datasets
- Score → evaluation mapping with `VLOOKUP`
- Nested conditional logic for complex calculations

### Week 2 — Print Layout & Pivot Tables

**Topics covered:**
- Print layout: margins, orientation, print area, page breaks
- Dynamic headers and footers
- Creating and managing Pivot Tables
- Filters, rows, columns, and values in Pivots
- Calculated fields and conditional formatting
- Pivot charts and scatterplots with trendlines

**Skills developed:**
- Making Excel sheets presentable and printable
- Summarizing complex datasets in a few clicks
- Creating multi-sheet reports with clean layouts
- Analyzing performance with dynamic filters

**Challenges overcome:**
- Sales analysis by category and store location
- Social media data integration and engagement calculation
- Complete reports with page-by-page insights

### Week 3 — Power Query & Data Transformation

**Topics covered:**
- Importing data from CSV and external tables
- Cleaning: rename columns, remove rows, fix formats
- Calculated columns: revenue, discounted prices, quarter, year-month
- Merge and append tables using common keys
- Grouping for totals and averages
- UnPivot and dataset restructuring

**Skills developed:**
- Building reusable transformation pipelines
- Handling datasets larger than traditional Excel limits
- Integrating multiple sources into a single dataset
- Checking data quality and distribution

**Challenges overcome:**
- Preparing clean, consistent AdventureWorks datasets
- Handling columns with spaces, wrong formats, duplicates
- Integrating product, sales, region, reseller, targets tables

### Week 4 — Power Pivot & Advanced Excel

**Topics covered:**
- Activating and using Power Pivot
- Data models with one-to-many relationships
- Hierarchies and slicers for data navigation
- DAX measures: dynamic rankings, top N, average comparisons
- Excel 365 functions: `XLOOKUP`, `FILTER`, `TEXTSPLIT`, `SEQUENCE`
- Advanced Pivot Tables on integrated sources

**Skills developed:**
- Transforming Excel into a complete BI tool
- Connecting tables in relational models
- Creating interactive dashboards with slicers and parameters
- Combining Power Query + Power Pivot for end-to-end analysis

**Challenges overcome:**
- Global dataset analysis with comparisons against world averages
- Dynamic rankings by selected indicators
- Final project: accommodation facilities and average prices

### Build Week — COVID-19 Analysis Italy 2020–2025

**Group project (Team F):**
- Integration of COVID datasets (Civil Protection) and vaccinations (Open Data)
- Cleaning and normalizing entities: dates, regions, fields
- Building KPIs: cases/recoveries, cases/deaths, cases/vaccinated
- Printable A4 dashboards with professional layout

**Deliverables:**
- Power Pivot model with dimension and fact table relationships
- Regional rankings 2020–2025 for cases, recoveries, deaths
- Percentage ratios for severity/effectiveness analysis
- 6 output sheets: Cases, Recoveries, Deaths, Mask, Ratios, Vaccine Effectiveness

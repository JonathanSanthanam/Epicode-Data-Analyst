# Week 12 — Python: Data Analysis, Visualization & Web Scraping

Questa settimana ha segnato la conclusione del modulo Python, concentrandosi sull'applicazione pratica dell'analisi dati. Ho imparato a effettuare una EDA (Exploratory Data Analysis) completa, a visualizzare i dati in modo efficace utilizzando librerie come Seaborn e Matplotlib, e ho esplorato le basi del Web Scraping per raccogliere dati dal web.

La settimana è culminata con il Progetto Finale, un'analisi approfondita sulla diffusione del COVID-19 nel mondo.

## Competenze sviluppate

- **Exploratory Data Analysis (EDA):** Utilizzo avanzato di Pandas per pulire, filtrare e aggregare grandi dataset
- **Data Visualization:** Creazione di grafici complessi (Boxplot, Histograms, Scatterplots, Heatmaps) con Seaborn e Matplotlib per comunicare insight
- **Gestione Date:** Conversione e manipolazione di serie temporali (datetime objects) per analisi trend
- **Web Scraping Basics:** Comprensione della struttura HTML/CSS e utilizzo di BeautifulSoup e requests per estrarre dati
- **Data Cleaning:** Gestione dei valori nulli (NaN), normalizzazione dei dati e preparazione per l'analisi

## Sfide risolte

- Scegliere il tipo di grafico più adatto per rappresentare relazioni tra variabili (es. Boxplot per distribuzioni, Scatterplot per correlazioni)
- Gestire dataset di grandi dimensioni (es. il dataset OWID Covid) e ottimizzare le query
- Confrontare metriche diverse (es. Terapia intensiva vs Ospedalizzazioni) tra nazioni specifiche (Italia, Germania, Francia)
- Interpretare la struttura del DOM di una pagina web per estrarre informazioni specifiche

# Week 12 — Python: Data Analysis, Visualization & Web Scraping

This week marked the conclusion of the Python module, focusing on the practical application of data analysis. I learned how to perform a complete EDA (Exploratory Data Analysis), visualize data effectively using libraries like Seaborn and Matplotlib, and explored the basics of Web Scraping to gather data from the web.

The week culminated in the Final Project, an in-depth analysis of the global spread of COVID-19.

## Skills developed

- **Exploratory Data Analysis (EDA):** Advanced use of Pandas to clean, filter, and aggregate large datasets
- **Data Visualization:** Creating complex charts (Boxplots, Histograms, Scatterplots, Heatmaps) with Seaborn and Matplotlib to communicate insights
- **Time Series Handling:** Converting and manipulating datetime objects for trend analysis
- **Web Scraping Basics:** Understanding HTML/CSS structure and using BeautifulSoup and requests to extract data
- **Data Cleaning:** Handling null values (NaN), normalizing data, and preparing datasets for analysis

## Challenges overcome

- Choosing the most appropriate chart type to represent relationships between variables (e.g., Boxplots for distributions, Scatterplots for correlations)
- Managing large datasets (e.g., the OWID Covid dataset) and optimizing queries
- Comparing different metrics (e.g., ICU vs. Hospitalizations) across specific nations (Italy, Germany, France)
- Interpreting the DOM structure of a webpage to extract specific information

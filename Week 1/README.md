Questa settimana ho studiato le funzioni logiche e di ricerca in Excel, concentrandomi su strumenti avanzati per l'analisi e manipolazione dei dati. Gli argomenti che ho approfondito includono:

Introduzione a Git: Ho aperto una repository per organizzare tutti gli esercizi e materiali
Teoria dei Grafici: Ho studiato i diversi tipi di grafici e quando utilizzarli

Ho applicato queste competenze per risolvere problemi pratici di analisi dati e ho iniziato a organizzare il mio lavoro attraverso il controllo di versione.

Conditional Counting and Summing: I used COUNTIF, SUMIF, SUMIFS for targeted analysis
Search Functions: I applied VLOOKUP, XLOOKUP, INDEX and MATCH to retrieve specific data
Text Functions: I used LEFT to extract portions of strings
VAT Calculations: I calculated VAT on taxable amounts and VAT-inclusive amounts for real scenarios
Git Introduction: I opened a repository to organize all exercises and materials
Chart Theory: I studied different chart types and when to use them

I applied these skills to solve practical data analysis problems and began organizing my work through version control.

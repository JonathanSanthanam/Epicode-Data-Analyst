Settimana 2 - Analisi Dati e Visualizzazione in Excel
Questa settimana si concentra sulle competenze pratiche di analisi dati utilizzando Excel, con particolare attenzione alla pulizia, manipolazione e visualizzazione dei dati. Gli argomenti principali includono:

Pulizia dei Dati: Rimozione di spazi, gestione di valori null e standardizzazione dei formati
Trasformazione dei Dati: Creazione di colonne calcolate, concatenazione di valori e applicazione di formattazione condizionale
Creazione di Grafici: Costruzione di diversi tipi di grafici (a barre, a torta) per rappresentare dati di vendita per categorie
Layout e Formattazione: Organizzazione dei fogli di lavoro con interruzioni di pagina, intestazioni e piè di pagina appropriati

Gli studenti lavorano con un dataset reale di inventario per praticare competenze essenziali di analisi dati, inclusa la preparazione dei dati, calcoli riassuntivi usando funzioni come SOMMA.SE, e creazione di visualizzazioni significative per comunicare insights aziendali.

Week 2 - Data Analysis and Visualization in Excel
This week focuses on practical data analysis skills using Excel, emphasizing data cleaning, manipulation, and visualization techniques. Key topics include:

Data Cleaning: Removing spaces, handling null values, and standardizing data formats
Data Transformation: Creating calculated columns, concatenating values, and applying conditional formatting
Chart Creation: Building multiple chart types (bar charts, pie charts) to represent sales data by different categories
Page Layout & Formatting: Organizing worksheets with proper page breaks, headers, and footers for professional presentation

Students work with a real inventory dataset to practice essential data analyst skills including data preparation, summary calculations using functions like SUMIF, and creating meaningful visualizations to communicate business insights.
